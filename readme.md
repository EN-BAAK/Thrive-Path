# 🌱 Thrive Path

A mobile application designed to help users improve their lives by creating and managing personal goals, tasks, and challenges in an organized and motivating way.

Built with:

* **React Native (Bare Workflow)**
* **TypeScript**
* **SQLite**

Repository: [https://github.com/EN-BAAK/Thrive-Path](https://github.com/EN-BAAK/Thrive-Path)

---

# ✨ Features

## 🎯 Goals Management

Create, manage, and track personal goals with progress monitoring.

## ✅ Tasks Management

Organize daily tasks and connect them to goals for better productivity.

## 🏆 Challenges System

Create personal challenges to stay motivated and build consistency.

## 🗂 Categories Management

Manage categories to organize goals, tasks, and challenges efficiently.

## 📱 Mobile Experience

Smooth and responsive mobile experience built with React Native.

---

# 🛠 Tech Stack

* React Native (Bare React Native)
* TypeScript
* React Navigation

# 🚀 Getting Started

## 1. Clone the Repository

```bash
git clone https://github.com/EN-BAAK/Thrive-Path.git
```

## 2. Navigate to the Project Folder

```bash
cd Thrive-Path
```

## 3. Install Dependencies

Using npm:

```bash
npm install
```

Or using yarn:

```bash
yarn install
```

---

# ▶️ Running the Project

## Start Metro Bundler

```bash
npm start
```

or

```bash
yarn start
```

---

## Run on Android

Make sure you have Android Studio and Android SDK configured.

```bash
npm run android
```

or

```bash
yarn android
```

# 📋 Requirements

Before running the project, make sure you have:

* Node.js
* npm or Yarn
* React Native CLI
* Android Studio / Android

Official React Native Environment Setup:
[https://reactnative.dev/docs/environment-setup](https://reactnative.dev/docs/environment-setup)
